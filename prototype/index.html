Notice

This is just a prototype of the Minimum Spanning Tree finder. In this version, even though the application/program asks you at the start how many vertices your graph has, the fact is that this program can only work with graphs with exactly 4 vertices at the time of writing. If you enter any number other than 4, it (the program) will return with an "error" in the form of a Javascript Alert (very "creative", I know). The full version of this application/program will (hopefully) be released in the future.

Furthermore, it has been observed that this program does not accept floats (i.e. decimal numbers) as arguments - this is due to the nature of some in-built Javascript functions. If I find a solution to fix it I will do so in the future.
